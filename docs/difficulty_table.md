# Dragon Jump 難度對應表 v1.2

## 分數段定義

### 0–199 新手保護區
- 主平台：白雲 90%、棕雲 10%，黑雲與磚塊不出現
- 間距：60–75px（畫面內約 8–10 個平台）
- 平台移動：全部靜止
- 敵人：不出現
- 待處理：a_haz and b_haz 分支可能漏出 BRICK，需將 score < 200 保護移至最前

### 200–299 入門挑戰
- 主平台：白雲 46%、棕雲 26%、黑雲 15%、磚塊 13%
- 間距：105–142px
- 平台移動：機率 60%，速度 130px/s
- 敵人：每 5 個平台一隻，靜止
- 待處理：間距從 75 跳到 105 落差過大，需加過渡段 80–100px

### 300–399 中級壓力
- 主平台：白雲 39%、棕雲 29%、黑雲 20%、磚塊 12%
- 間距：140–142px
- 平台移動：機率 75%，速度 155px/s
- 敵人：每 4 個平台一隻，移動 60px/s

### 400+ 高手模式
- 主平台：白雲 32%、棕雲 32%、黑雲 30%、磚塊 6%
- 間距：140–142px
- 平台移動：機率 85%，速度 220px/s
- 敵人：每 3 個平台一隻，移動 100px/s
